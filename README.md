# open-sass

A modern SaaS starter template with everything you need to build and ship fast.

## Stack

- **Framework**: Next.js 16 + React 19 + TypeScript
- **Database**: PostgreSQL + Drizzle ORM
- **Auth**: Better Auth (email/password)
- **Payments**: Stripe (subscriptions)
- **UI**: shadcn/ui + Tailwind CSS v4
- **Email**: Resend
- **Dev**: Docker Compose + Makefile

## Quick Start

```bash
# Clone the template
gh repo create my-saas --template YOUR_USERNAME/open-sass
cd my-saas

# Run setup
./setup.sh

# Start developing
make dev
```

## Setup

The `setup.sh` script will:

1. Ask for your project name and details
2. Generate auth secrets
3. Replace all placeholders
4. Create your `.env` file
5. Install dependencies

## Development

```bash
# Start everything (database + dev server)
make dev

# Just start the database
make db

# Stop all services
make down

# Clean everything
make clean
```

## Environment Variables

Copy `.env.example` to `.env` and fill in:

- `DATABASE_URL` - PostgreSQL connection string
- `BETTER_AUTH_SECRET` - Generated by setup script
- `STRIPE_SECRET_KEY` - From Stripe dashboard
- `STRIPE_PUBLISHABLE_KEY` - From Stripe dashboard
- `STRIPE_WEBHOOK_SECRET` - From Stripe CLI or dashboard
- `STRIPE_PRICE_*` - Your Stripe price IDs
- `RESEND_API_KEY` - From Resend dashboard

## Stripe Setup

1. Create products in your Stripe dashboard
2. Copy the price IDs to your `.env`
3. Run `make stripe-listen` for local webhook testing

## Releasing

```bash
pnpm run release patch  # 0.1.0 -> 0.1.1
pnpm run release minor  # 0.1.0 -> 0.2.0
pnpm run release major  # 0.1.0 -> 1.0.0
```
